1. Nombre de la empresa
Fight and Fitness Center 

2. Descripción del negocio
El centro de entrenamiento físico “Fight and Fitness Center” ofrece un entrenamiento integral a la comunidad. Incluye sala de musculación, clases grupales de crossfit y entrenamientos personalizados de artes marciales.

3. Necesidad digital identificada
Se necesita una página web informativa para dar a conocer el gimnasio, sus especialidades, clases y planes de membresía.  
También debe permitir que los usuarios puedan contactarse fácilmente para tomar membresias o hacer consultas, facilitando la captación de nuevos alumnos.

4. Secciones del sitio web
- Inicio (Banner principal)
- Sobre nosotros
- Servicios / Clases
- Membresías
- Contacto (formulario, mapa y redes sociales)
